# AssetFlow - Enterprise Asset & Resource Management System (ERP)

Welcome to **AssetFlow**, a centralized ERP platform for organizations to manage physical assets, shared resources, departments, employees, bookings, maintenance, and audit trails.

## Project Structure

```text
AssetFlow/
├── frontend/    # React (Vite, TS) Client Application (Phase 1)
└── backend/     # Express API (NodeJS, Postgres) Server Application (Future Phase)
```

## Running the Application

### Frontend Client

To run the frontend developer server locally:

1. Navigate to the `frontend` folder:
   ```bash
   cd frontend
   ```
2. Install the node packages:
   ```bash
   npm install
   ```
3. Run the development build:
   ```bash
   npm run dev
   ```

To build the static application for production:
```bash
npm run build
```
